# Git Commands

```bash
# Create a git Repo
git init


# Add all the untrack file to staging
git add .

# Adding a individual file to staging
git add  index.js

# added comment for the particular commit
git commit -m "Your ugly comment here"


# create new branch from the current branch
git checkout -b name_of_new_branch

# jumping to other branch or particular commit hash
git checkout name_of_branch_or_commit_hash


# Setting up stream -- Connecting your local branch to new remote branch
git push -u origin name_of_current_branch

# push your code to server
git push


# pull new code to local from remote.
git pull
```
